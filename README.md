# Lait Pur

Премиальный сайт-портфолио органического молока на Next.js 16, React 19, Tailwind CSS 4 и Framer Motion.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Секции

- **Hero** — бренд Lait Pur, letter-reveal, 3D-бутылка с параллаксом
- **Our Story** — асимметричная Bento-сетка
- **The Milk Bar** — переключатель вкусов со сменой фона
- **Cow-to-Glass** — таймлайн, заполняющийся при скролле
- **Footer** — рассылка с анимацией наливания

## Сборка

```bash
npm run build
npm start
```
